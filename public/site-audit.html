Cognitive Load Management

Progressive Disclosure

Persona intake captures user type before showing relevant features. Chat reveals complexity gradually.

Single Responsibility Views

Each page has one purpose. No mixed editing/viewing states. Modals for secondary actions.

Consistent Patterns

All cards use same border radius, spacing, and interaction patterns. Buttons follow a strict hierarchy.

Reduced Decision Fatigue

Pricing uses audience tabs to show only relevant plans. Feature pages group by use case.

Clear Visual Hierarchy

H1 > H2 > H3 with consistent sizing. Primary CTAs teal, secondary outlined, destructive red.

Information Chunking

Legal content broken into collapsible sections. Long forms use multi-step flows.

Contextual Help

Tooltips and inline explanations for legal terms. "What does this mean?" links throughout.

Error Prevention

High-risk acknowledgment modal before sensitive tools. Jurisdiction gating prevents irrelevant results.
